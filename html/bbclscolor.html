bbClsColor red,green,blue

Arguments
redamount of red (0..255)
greenamount of green (0..255)
blueamount of blue (0..255)
Description

The bbClsColor command is used to change the Color used by the bbCls command.

See the bbColor command for combining values of red, green and blue in order to specify some commonly used colors.

Interfaces
BBSDK_API void bbClsColor( int r,int g,int b );
public static int bb.ClsColor(int red,int green,int blue)
