bbEntityX# ( entity[,globalspace] )

Arguments
entityentity handle
globalTrue
Description

Returns the X component of the entities current position in local or optionally global space.

The X axis is tradionally the left to right axis in Blitz3D space when facing towards Z either from the parents point of view or optionally from the center of the world.
