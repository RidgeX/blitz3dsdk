bbModule Installation

To install the Blitz3DSDK BlitzMax module copy the blitz3d.mod module group folder from the Blitz3DSDK/BlitzMax folder to your $BlitzMax/mod directory. The blitz3d.mod folder should be located alongside the brl.mod and pub.mod module folders and contains the actual blitz3dsdk.mod module.

Once the blitz3d.mod module group has been copied use the docmods feature in the MaxIDE Tools menu to include the documentation included with the Blitz3DSDK module.

Links to samples and games can be found in the newly created Blitz3DSDK chapter of the help section of the MaxIDE navbar.
