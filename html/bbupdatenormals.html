bbUpdateNormals mesh

Arguments
meshmesh handle
Description

Recalculates all normals in a mesh.

This is necessary for correct lighting if you are building or modifying meshes and have not set surface normals manually using the bbVertexNormal commands or a mesh has been loaded from a model file with bad or missing vertex normal data.
