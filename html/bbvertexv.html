bbVertexV# ( surface,index[,coord_set] )

Arguments
surfacesurface handle
indexindex of vertex
coord_setoptional UV mapping coordinate set. Should be set to 0 or 1.
Description

Returns the texture v coordinate of a vertex.

The coord_set defaults to 0 but may optionally be 1 to specify the value returned refer to the vertex's secondary texture coordinate set.
