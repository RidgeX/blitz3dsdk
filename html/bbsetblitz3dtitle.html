bbSetBlitz3DTitle ( windowtitle$[,quitmessage$] )

Description

Specifies a title for any windows created by the Blitz3DSDK and optionally a message that is displayed when the user closes a window created by the Blitz3DSDK.

Interfaces
BBSDK_API void bbSetBlitz3DTitle(const char *title,const char *close);
public static int bb.SetBlitz3DTitle(String windowtitle,String quitmessage)
