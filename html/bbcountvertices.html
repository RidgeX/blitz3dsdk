bbCountVertices ( surface )

Arguments
surfacesurface handle
Description

Returns the number of vertices in a surface.

Use the result of bbCountVertices command to make sure your program only modifies vertices that exist. Vertex modifier commands such as bbVertexCoords, bbVertexColor, bbVertexNormal and bbVertexTexCoords all use a vertex index parameter that should be in the range of 0..CountVertices()-1.
