bbWaitMouse ( )

Description

bbWaitMouse causes the program to halt until a mouse button is pressed by the user and returns the ID of that button.

bbWaitMouse will wait and return 1 for the left button, 2 for the right and 3 for the middle button when pressed.

Interfaces
BBSDK_API int bbWaitMouse();
public static int bb.WaitMouse()
