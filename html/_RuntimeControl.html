Runtime Control

bbBeginBlitz3D
Initializes the Blitz3D runtime environment and returns 0 if there is an error.
bbEndBlitz3D
Closes down the Blitz3D runtime environment and releases all resources.
bbRuntimeError
Closes down the Blitz3D runtime environment and causes the Blitz3D runtime handler to be called which by default displays the message and ends.
bbSetBlitz3DDebugMode
Sets the debugging level for the Blitz3DSDK.
bbSetBlitz3DDebugCallback
Installs an error handler for use by the Blitz3DSDK.
bbSetBlitz3DEventCallback
Installs an event handler for use by the Blitz3DSDK.
bbSetBlitz3DHWND
Specifies a parent window for the Blitz3DSDK to use with the windowed modes of the bbGraphics3D command.
bbSetBlitz3DTitle
Specifies a title for any windows created by the Blitz3DSDK and optionally a message that is displayed when the user closes a window created by the Blitz3DSDK.
