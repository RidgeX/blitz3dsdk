About

The Blitz3D SDK provides an interface to the popular Blitz3D game engine for BlitzMax, C++ and C# programmers.

Commands are provided for programming all aspects of a typical video game including 2D and 3D Graphics, Sound and User Input.

The Blitz3D SDK enables the programmer to concentrate on developing their game by extending their programming language of choice with a command library that is both concise but powerful.
