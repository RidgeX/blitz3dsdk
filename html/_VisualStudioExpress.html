Visual Studio Express

The Blitz3DSDK distribution contains files to work with both Microsoft's .Net managed languages including a C# definition file with embedded quickhelp and C++ compatible library and header files.

All Microsoft Visual Studio Express languages are currently free versions of their professional products and with the addition of the Blitz3DSDK provide an excellent introduction to industrial strength C++ and C# development.

Begin by installing either the C++ or C# Express editions:

http://msdn.microsoft.com/vstudio/express/downloads/

After installation begin by loading the example projects that are included in the Blitz3DSDK distribution and then check the following chapters for creating new Blitz3DSDK projects in either Visual C++ or Visual C#.
