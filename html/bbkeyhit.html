bbKeyHit ( scancode )

Arguments
scancodescancode of key to test
Description

Returns the number of times the specified key has been pressed since the last time the bbKeyHit command was called with the specified scancode.

bbKeyHit will only return positive once when a key is pressed where as bbKeyDown will repeatedly return True until the specified key is released.
