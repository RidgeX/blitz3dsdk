bbJoyUDir ( [port] )

Arguments
portnumber of joystick port to check (optional)
Description

Returns an integer value of -1, 0 or 1 representing the horizontal direction of the joystick's second stick be it left, centered or right.

ValueDirection
-1left
0centered
1right
