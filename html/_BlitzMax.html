BlitzMax

Module Installation

To install the Blitz3DSDK BlitzMax module copy the 'blitz3d.mod' folder from the SDK to your blitzmax 'mod' folder. The blitz3d.mod folder should be located alongside the brl.mod and pub.mod module folders and contains the actual blitz3dsdk.mod module.

Once the blitz3d.mod module group has been copied use the docmods feature in the MaxIDE Tools menu to include the documentation included with the Blitz3DSDK module.

Links to samples and games can be found in the SDK 'samples' folder.

Simple Example

The following is a simple BlitzMax application featuring a rotating cube using the Blitz3DSDK module. 

Import blitz3d.blitz3dsdk

Strict

bbBeginBlitz3D

bbGraphics3D 800,600,0,2

Local cube=bbCreateCube()
Local light=bbCreateLight()
Local camera=bbCreateCamera()

bbPositionEntity camera,0,0,-4

While Not bbKeyHit(1)
	bbTurnEntity cube,1,2,3
	bbRenderWorld
	bbFlip 1
Wend
