bbGetColor x,y

Arguments
xhorizontal pixel position
yvertical pixel position
Description

The bbGetColor command sets the current drawing color to that of the pixel at the specified screen coordinates.

The bbColorRed, bbColorGreen and bbColorBlue functions can be used to retrieve the current drawing color, allowing you to determine a pixel's color.

Interfaces
BBSDK_API void bbGetColor( int x,int y );
public static int bb.GetColor(int x,int y)
