bbClearSurface surface,[clear_verts,clear_triangles]

Arguments
surfacesurface handle
clear_vertsTrue
clear_trianglesTrue
Description

Removes all vertices and / or triangles from a surface.

The two optional parameters clear_verts and clear_triangles default to True
