bbWaitJoy ( [port] )

Arguments
portoptional joystick port to pause for
Description

Waits for any joystick button to be pressed and returns the button identifier.

bbWaitJoy causes the program to pause until any button of the specified joystick is pressed.

If there is no gaming device connected or the optional port identifier is not a valid device bbWaitJoy will not pause but return 0 immediately.
