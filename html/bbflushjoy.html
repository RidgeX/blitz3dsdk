bbFlushJoy

Description

Resets the state of the internal joystick button map so all buttons of all joysticks are considered up and all joystick events are discarded.

bbFlushJoy is useful when transitioning from control systems based on the state commands such as bbJoyDown to an event style control using the bbGetJoy command and any buffered button presses need to be discarded.
