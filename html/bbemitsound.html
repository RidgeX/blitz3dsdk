bbEmitSound ( sound,entity )

Arguments
soundsound handle
entityentity handle
Description

Emits a sound attached to the specified entity and returns a sound channel.

The sound must have been loaded using bbLoadSound with the SOUND3D flag.

The sound channel returned can subsequently be used with such sound channel commands as bbChannelVolume and bbChannelPitch.
