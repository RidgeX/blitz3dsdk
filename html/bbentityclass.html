bbEntityClass$ ( entity )

Arguments
entitya valid entity handle
Description

Returns a string containing the class of the specified entity.

Possible return values are "Pivot", "Light","Camera", "Mirror", "Listener", "Sprite", "Terrain", "Plane", "Mesh", "MD2" and "BSP".

Note that bbEntityClass function will fail if a valid entity handle is not supplied and will not just return an empty string.
