bbSetBlitz3DRuntime ( gxruntime )

Description

Installs a third party rendering engine.

bbSetBlitz3DRuntime should be called before bbBeginBlitz3D with the handle created by the third party Blitz3D renderer.

Interfaces
BBSDK_API void bbSetBlitz3DRuntime(void *gx);
public static int bb.SetBlitz3DRuntime(int gxruntime)
