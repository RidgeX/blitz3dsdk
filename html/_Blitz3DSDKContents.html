Blitz3DSDK Contents

The Blitz3DSDK is distributed as a single zip archive.

To install open the archive and drag it's contents into a folder, typically in the same location you currently save your programming projects.

Files

The Blitz3DSDK contains the following files and directories:

redist\b3d.dll

The Blitz3DSDK runtime library. Blitz3DSDK based applications will need to be able to access this file to run. You should place this file either in the same directory as your application executable, or in the windows/system32 directory.

lib

This directory contains the file blitz3dsdk.lib, the Blitz3DSDK static link library for use with C/C++ applications. C/C++ applications will need to link with this library.

include

This directory contains the files blitz3dsdk.h, blitz3dsdk.cs and blitz3dsdk.pbi. These are header files for the languages C/C++, C# and PureBasic, respectively.

blitz3d.mod

This is Blitz3D in module form for BlitzMax. Simply copy this folder into your blitzmax 'mod' directory and you can start using Blitz3D in BlitzMax!
