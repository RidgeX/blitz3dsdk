bbVectorYaw# ( x#,y#,z# )

Arguments
x#x vector length
y#y vector length
z#z vector length
Description

Returns the yaw value of a vector.

Yaw is a common name for rotation around the Y axis or in this instance the compass heading in degrees if z is north and x is west.
