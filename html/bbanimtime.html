bbAnimTime# ( entity )

Arguments
entityentity handle
Description

Returns the current animation time of an entity.

bbAnimTime returns a float between 0.0 and the value returned by the bbAnimLength() function unless the animtion is in transition in which case a value of 0.0 is returned.
