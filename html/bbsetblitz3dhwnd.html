bbSetBlitz3DHWND ( hwndparent )

Description

Specifies a parent window for the Blitz3DSDK to use with the windowed modes of the bbGraphics3D command.

bbSetBlitz3DHWND must be called before bbBeginBlitz3D.

Call bbGraphics3D using the GFX_WINDOWED mode and the maximum size of the parent window and use the bbViewport and bbCameraViewport commands to configure the 3D display to the parent window's current size.
