Distribution Details

The Blitz3DSDK is distributed as a single zip archive.

To install open the archive and drag it's contents into a folder, typically in the same location you currently save your programming projects.

Files

The Blitz3DSDK contains the following files and directories:

blitz3dsdk.dll

The Blitz3DSDK runtime. Distribute the blitz3dsdk.dll file with your program files.

lib

Contains the blitz3dsdk.lib library file required to link with C++ projects.

include Contains the blitz3dsdk.h header file required to compile C++ projects.

classes

Contains the blitz3dsdk.cs csharp class for C# projects.

BlitzMax

Contains the blitz3dsdk.mod command module and example programs for BlitzMax programmers.

BiltzMax users should copy the blitz3dsdk.mod folder into the $blitzmax/mod/brl.mod folder of their current BlitzMax installation.

DevC

Blitz3DSDK templates for building apps with Bloodshed's DevC++

PureBasic

Contains the Blitz3DSDK pbi interface file for the Pure Basic language and a collection of examples.

VisualC

Example project and blank project for VisualC++ development

VisualCSharp

Example of using Blitz3DSDK in .net environment
