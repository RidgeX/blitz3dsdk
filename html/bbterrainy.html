bbTerrainY# ( terrain,x#,y#,z# )

Arguments
terrainterrain handle
x#world x coordinate
y#world y coordinate
z#world z coordinate
Description

Returns the interpolated y coordinate on a terrain.

bbTerrainY can be used to calculate the effective height of a terrain directly below / above the specified point.
