bbEntityVisible ( src_entity,dest_entity )

Arguments
src_entitysource entity handle
dest_entitydestination entity handle
Description

Returns True

This command casts a ray (an imaginary line) from src_entity to dest_entity. If the ray hits an obscurer entity the result is False otherwise the function returns True

See the bbEntityPickMode for setting an entity as an obscurer.
