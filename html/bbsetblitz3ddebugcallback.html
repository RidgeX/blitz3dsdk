bbSetBlitz3DDebugCallback ( callback )

Description

Installs an error handler for use by the Blitz3DSDK.

The prototype for the callback is 

void (*errorhandler)(const char *errormessage));

The Blitz3D runtime is closed before the error handler is invoked by the Blitz3DSDK.

The BlitzMax Blitz3DSDK module installs a default error handler which throws a BlitzMax RuntimError containing the error message.
