bbAddTriangle ( surface,v0,v1,v2 )

Arguments
surfacesurface handle
v0index number of first vertex of triangle
v1index number of second vertex of triangle
v2index number of third vertex of triangle
Description

Returns the index of a new triangle added to the specified surface.

The three vertex indexes define the points in clockwise order of a single sided triangle that is added to the surface specified.

The v0, v1 and v2 parameters are the index numbers of vertices added to the same surface using the AddVertex function.

A special DoubleSided effect can be enabled for a surface that will treat each triangle in a surface as having two sides, see bbEntityFX and bbBrushFX for more information.
