bbGetKey ( )

Description

Returns an ascii code corresponding to the key last typed by the user or 0 if all keyboard events have been reported.

As Blitz uses an internal queue for tracking key presses it is sometimes useful to call bbGetKey multiple times to empty the queue. Use the bbFlushKeys command to completely empty the internal key queue.

The bbGetKey function is useful for situations when the user is expected to type some text.

The bbKeyDown function is more appropriate when the player is expected to hold down certain key combinations in a more action oriented game environment.
