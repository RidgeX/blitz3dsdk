bbLoadSprite ( path$[,tex_flag][,parent] )

Arguments
path$filename of image file to be used as sprite
tex_flagoptional texture flag
parentoptional parent of entity
Description

Creates a sprite entity, and assigns a texture to it.

NameValueDescription
TX_COLOR1Color (default)
TX_ALPHA2Alpha
TX_MASKED4Masked
TX_MIP8Mipmapped
TX_CLAMPU16Clamp U
TX_CLAMPV32Clamp V
TX_SPHERE64Spherical reflection map
TX_CUBIC128Cubic environment map
TX_VRAM256Store texture in vram
TX_HIGHCOLOR512Force the use of high color textures

See the bbCreateTexture command for a detailed description of the texture flags.

The optional parent parameter attaches the new Sprite entity to a specified parent entity. See the bbEntityParent command for more information on entity hierachy.
