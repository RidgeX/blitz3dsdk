Entities

The commands in the following section contain a collection of generic operations that can be used with all entity types as well as those specific to certain types.

All entitity types have a 3D position, direction and scale that is defined relative to their parent.

They all have a display state that can be switched on and off using the bbHideEntity and bbShowEntity commands.

Brush, alpha and fx properties are available at the Entity level that combine with each of the respective surface properties contained in the Entity at rendertime.

Collision and picking commands allow all entities to provide interactive responses to either each other or rays fired by the various pick methods provided.

bbEmitSound can be used to turn any entity into a sound source.
