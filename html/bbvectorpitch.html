bbVectorPitch# ( x#,y#,z# )

Arguments
x#x vector length
y#y vector length
z#z vector length
Description

Returns the pitch value of a vector.

Pitch is a common name for rotation around the X axis or in this instance the angle the vector is raised if y is up and the distance x,z is forwards.
