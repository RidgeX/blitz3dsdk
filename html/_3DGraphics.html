3D Graphics

The Blitz3D command set is a wealth of functions for the control and rendering of 3D scenes.

For inspiration to what is possible check out the gallery at http://blitzbasic.com/gallery/gallery.php where programmers since 2001 have been sharing images of the worlds being created with the Blitz3D command set.

The bbGraphics3D command creates a 3D display at the resolution specified resetting the Blitz3D scene graph and texture buffers and hence should be the first command called before using any commands found in the 3D sections of this documentation.

Once the world is initialized various commands are available to create 3D objects known as entitys.
