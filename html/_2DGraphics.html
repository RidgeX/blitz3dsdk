2D Graphics

The Blitz3DSDK includes support for 2D graphics development including the ability to draw text, shapes and display images in a variety of file formats in either a windowed or fullscreen display.

ExtensionCompressionFeatures
bmp image none can be created with bbSaveImage command
png image good 32 bit loss-less compression
jpg image excellent small loss in image quality
tga image none 32 bit

The 2D commands in the following section can be used with a plain bbGraphics based display or a 3D display initialized with the bbGraphics3D command.

The advantages of using a plain bbGraphics display when 3D graphics are not required includes being compatible with older generation computers that do not feature 3D accelerated graphics cards.

Other commands provided feature the ability to read and write the pixel buffers of both display, image and texture buffers.

Collision testing functions are provided that make testing for overlapping images and rectangles easy.
